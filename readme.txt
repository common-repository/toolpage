== Description ==

**ENGLISH**

Create unlimited Landing Pages for your OnLine Advertising.
ToolPage is a plugin to manage one or more landing pages. You have more Template-scheme based on elements called Box. **You can set a ToolPage as Homepage/Front Page.**

A **ToolPage** is composed of one or more **Box**. A Template is composed of one or more columns.

General Infos of a **ToolPage**:

* 	NEW: Appearance Themes Base and Professional
* 	NEW: Admin toolbar links
* 	NEW: Widget
* 	Box Type: text / html, image, menus, forms, twitter, facebook, youtube
* 	Customizing CSS: background color, font, font color, font size
* 	Custom CSS-style
* 	SEO Friendly URL
* 	SEO Optimization
* 	Google Analytics

**Required**

*	Permalinks activated

== Installation ==

**NEW!** **NEW!** Required: Permalinks activated

Method 1

* 	Go to WordPress admin -> Plugins -> Add new
* 	Search "**toolpage**"
* 	Click on "**Install now**"
* 	Activate the plugin, click on "**Activate Plugin**"
* 	Go to Settings -> **Permalinks** and click on **Save Changes**
* 	Installation Help & Support: <a href="http://www.make23.com/soluzioni/toolpage/">ToolPage Support</a>

Method 2

* 	Download the zipped plugin file to your local machine.
* 	Go to WordPress admin -> Plugins menu
* 	Add New->Upload->browse the zipped plugin
* 	Upload and Activate
* 	Go to Settings -> Permalinks and click on **Save Changes**
* 	Installation Help & Support: <a href="http://www.make23.com/soluzioni/toolpage/">ToolPage Support</a>

Method 3

* 	Download the zipped plugin file to your local machine.
* 	Unzip the file.
* 	Upload the "toolpage" folder to the "/wp-content/plugins/" directory. (DO NOT CHANGE ANY FILE OR FOLDER NAMES)
* 	Activate the plugin through the "Plugins" menu in WordPress.
* 	Go to Settings -> Permalinks and click on **Save Changes**
* 	Installation Help & Support: <a href="http://www.make23.com/soluzioni/toolpage/">ToolPage Support</a>
